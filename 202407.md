    # 扣子 API 知识库文件上传规则说明
    
    ## AI摘要
    该接口用于向指定知识库上传文件，每次最多 10 个，要匹配知识库类型，如 txt 文档只能到文档库。每次请求仅一种上传方式，不支持同时上传在线网页和本地文档，知识库分段策略依首次上传设置为准。备注为“扣子 API 文档”。
