    # 扣子 API 知识库文件上传规则说明
    
    ## AI摘要
    该接口用于向指定知识库上传文件，每次最多 10 个，要匹配知识库类型，如 txt 文档只能到文档库。每次请求仅一种上传方式，不支持同时上传在线网页和本地文档，知识库分段策略依首次上传设置为准。备注为“扣子 API 文档”。

    ## 扣子 Bot 模式：单与多 Agent 之析
    
    ### AI摘要
    在扣子中创建 Bot 后，单 Agent 模式处理复杂任务时，需详细冗长提示词，加插件和工作流，调试复杂且易影响整体功能和结果。为解决此，扣子提供多 Agent 模式，可添加多个 Agent 并配置节点分工协作，高效处理复杂用户任务。
